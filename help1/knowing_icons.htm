Knowing icons
Home
Takes you back to the homepage
Orphan
Gives you a view of Orphan table/page including all kind of operations you might need to perform pertaining to Orphans.
This is the main table and is connected to 5 other tables. Add new, view all , search and back
Adoption
Gives you a view of Adoption table including all kind of operations you might need to perform. Add new, view all, search and back
Donor
Gives you a view of Donor table including all kind of operations you might need to perform. Add new, view all, search and back
Appointment
Links you to an appointments calendar that allows you to set appointments as well as other relating fields in connection to them like location
Volunteer
Gives you a view of Volunteer table including all kind of operations you might need to perform. Add new, view all, search and back
Administrator
(Found in Admin Side Only)
Gives  you a view of the user admintable that allows you to administer users of the application by either create new users or just setting passwords
Help
Gives a brief insight of the application and of all the functions that can be performed
Add
Allows for you to add a record to a table
View
Allows you to view a tableâ€™s contents
Search
Allows for one to search a particular table using a search term in an input box located to the left of this symbol by particular fields like First Name
General Button
Will generally have different messages and will give you access to several views and tables
Handler Icon
This icon refers to the view for the table relating to Handler. Who is usually perceived as the first contact of the orphan to the orphanage.
 
Hospital
This icon relates the view for the table of hospital records.  Basic information pertaining to an orphans medical information is kept here
School Icon
This icon relates to the view of the school records table. It relates to all the orphans school information.
Next of Kin Icon
This icon relates to the view of the Relations or Next of Kin tables that hold information on the closest living relatives of the orphans
 
Case Icon
The case icon relates to the view of all the current cases that may pertain to a particular orphan under the care of a social worker
Social Worker
This icon relates to the view of social workers who are government officials that work with the orphanage to provide the orphans with a safe environment for them to develop.
Adopters
This icon relates to the view of the adopters or prospective parents. The adoption process requires the details of the main adopter and possibly a spouse.
